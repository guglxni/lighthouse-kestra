# Graphify outputs (generated)

Running `./scripts/graphify_lighthouse.sh` or `make graphify` fills `output/` with Graphify’s default artifacts (for example `graph.html`, `GRAPH_REPORT.md`, `graph.json`). The script also copies `graph.html` to `report.html` for predictable naming.

Staging files under `.staging/` are ephemeral and gitignored.

See the root [README.md](../../README.md) section **Architecture graph (Graphify)** for install and API-key requirements.
